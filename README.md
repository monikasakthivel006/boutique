# 🧵 Monika Tailoring Boutique Website

A modern and responsive tailoring boutique website built using **HTML** and **CSS**. This project showcases a professional tailor shop with an attractive user interface and essential sections for customers.

## ✨ Features

- 🏠 Home Page
- 👩 About Us
- ✂️ Services
- ⭐ Why Choose Us
- 🖼️ Gallery
- 💰 Price List
- 📅 Appointment Booking Form
- 💬 Customer Reviews
- 📞 Contact Section
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
